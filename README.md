# REACT-REALTIME-CHAT

## Description

This is a simple chat application built with Typescript, React, Node.js, Express and Socket.io.

## Installation

Head to server and client folders and run `npm install` to install all dependencies.

## Usage

> Inside server and client

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Production

```bash
npm run start
```
